# Pivot Tables

Generate cross-tabulated reports of your Jira data for quick analysis.

## Creating a Pivot Report

The **Pivot Table** feature (table icon ⊞) creates a dynamic summary of your data on a new sheet.

1. Select your **Rows**: e.g., "Assignee" to see distribution by person.
2. Select your **Columns**: e.g., "Status" to see workload state.
3. Select your **Value**: e.g., "Issue Count" or total "Story Points".
4. (Optional) Apply a **User Filter** to focus on a specific developer.
5. Click **Generate Pivot Table**.

## Customizing the View
- **Jira Pivot Sheet**: A new sheet is created with formatted cells and totals.
- **Conditional Formatting**: High-volume cells are highlighted to help you spot bottlenecks or overallocation.
