# Marketplace Assets

This folder contains the required image assets for publishing "Jira Sync" to the Google Workspace Marketplace.

## Required Icons
- `icon-128x128.png`: **Application Icon** (128x128 px). Used for the store listing and identifying the app.
- `card-banner-440x280.png`: **Card Banner** (440x280 px). Used for promotional cards in the Marketplace.

## Screenshots (1280x800 px)
These screenshots demonstrate the add-on functionality within Google Sheets, as required by the Marketplace.
- `screenshot-config-1280x800.png`: Shows the **Configuration** tab where users set up their Jira connection.
- `screenshot-sync-1280x800.png`: Shows the **Sync Results**, displaying populated data in the sheet.
- `screenshot-issues-1280x800.png`: Shows the **Issues Manager** tab for JQL queries and bulk actions.
- `screenshot-roadmap-1280x800.png`: Shows the **Roadmap Planning** feature with timeline visualization.

## Note
- For the **32x32 px** icon requirement, you can resize `icon-128x128.png` using any image editor (Preview on Mac, or online tools). Google often accepts the larger file and resizes it automatically, but have a resized version ready just in case.
