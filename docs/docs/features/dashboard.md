# Dashboard & Metrics

Visualize your project health instantly with the Dashboard generator. Turn raw data into actionable insights for your team meetings and stakeholder updates.

![Dashboard Interface](/img/screenshot-dashboard.png)

## Creating a Dashboard

1. Navigate to the **Dashboard** tab in the sidebar (pie chart icon).
2. Select the data columns you want to visualize.
   - **Status**: See progress breakdown (Pie Chart).
   - **Priority**: Identify urgent tasks (Bar Chart).
   - **Assignee**: Balance workload (Column Chart).
3. Click **Generate Dashboard**.
4. A new sheet named **"Jira Dashboard"** will be created with interactive charts and pivot tables.

## Customizing Your View

The generated dashboard is a standard Google Sheet. You can:
- **Resize charts**: Drag corners to adjust size.
- **Change chart types**: Edit chart settings to switch from Pie to Bar, etc.
- **Refresh data**: Click **Generate Dashboard** again to update with the latest synced data.

:::tip
Always save a copy of your favorite dashboard layout before regenerating, as the generator will overwrite the "Jira Dashboard" sheet.
:::
